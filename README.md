# Personal Blog

Personal blog application built with Go.

The project provides a backend service for managing blog posts, user content, and basic blog functionality while demonstrating Go backend development principles.

## Features

- Create blog posts
- Read blog posts
- Update blog posts
- Delete blog posts
- REST API architecture
- Backend routing
- JSON request/response handling
- Structured project organization

## Tech Stack

- Go
- REST API
- JSON
- HTTP Server

## Installation

```bash
go mod download
```

## Run Project

```bash
go run main.go
```

## Project Structure

- Handlers
- Routes
- Models
- Business logic
- HTTP server

## Purpose

The project demonstrates backend development with Go, API design, and server-side application architecture.
